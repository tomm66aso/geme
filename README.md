# Da i Gemelli — Premium Vercel Build

Pronto per GitHub/Vercel.

## Deploy
1. Carica questi file nella root del repository GitHub.
2. Importa il repo su Vercel.
3. Framework preset: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
